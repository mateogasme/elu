# @mgasme/elu-core

Framework-agnostic runtime for [elu](https://github.com/mateogasme/elu). Ships the controller, keyboard matrix, shared types, and CSS.

This package is an internal shared dependency consumed by `@mgasme/elu` (Astro), `@mgasme/elu-react` (React), and `@mgasme/elu-vue` (Vue 3). Install one of those packages instead.
