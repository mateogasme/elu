# Changelog

All notable changes to `@mgasme/elu` (Astro) are documented here.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and the project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.4.0] - 2026-05-15

### Changed
- Version bump to align with `@mgasme/elu-core`, `@mgasme/elu-react`, and the new `@mgasme/elu-vue` 0.4.0.

## [0.3.1]

Concentric border-radius fixes and minor visual polish.

## [0.3.0]

`SectionExamples` content surfaced on the landing — workspace switcher, model picker, assignee picker.

## [0.2.x]

Runtime extracted into `@mgasme/elu-core`. `src/{controller,keyboard,types}.ts` became one-line shim re-exports. Behavior unchanged.

## [0.1.0]

Initial Astro release. Compound-component API, native Popover + CSS Anchor Positioning, Firefox fallback, APG combobox + listbox accessibility, theming via `--elu-*` CSS variables scoped to `[data-elu-root]`, form integration via hidden inputs.
